| 문제 | 잠재적 원인 | 해결 방법 |
|:---------------------------|:------------------------------|:----------------------------------------------------------------------|
| **게임 튕김/멈춤** | RAM 부족, 과열 | 그래픽 설정 낮추기, 백그라운드 앱 종료, 기기 냉각. |
| **성능 저하/지연(Lag)** | 낮은 CPU/GPU 성능, 불안정한 인터넷 | 그래픽 낮추기, 안정적인 Wi-Fi 확인, 기기 캐시 삭제. |
| **다운로드 오류** | 저장 공간 부족, 네트워크 문제 | 공간 확보 (30GB 이상), 인터넷 연결 확인, 다운로드 재시작. |
| **로그인 문제** | 잘못된 계정 정보, 서버 문제 | 넷마블 ID/비밀번호 재확인, 서버 상태 확인, 앱 재시작. |
| **그래픽 깨짐** | 오래된 드라이버, 호환되지 않는 GPU | 안드로이드 OS 업데이트, 게임 최신 상태 확인, 렌더링 옵션 조정. |
