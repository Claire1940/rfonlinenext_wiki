### 인게임 그래픽 설정 조정

검은 화면에도 불구하고 잠시 게임에 접속하거나 오디오를 들을 수 있다면 그래픽 설정을 조정해 보십시오.

| 설정 카테고리    | 권장 변경 사항                                      | 검은 화면에 미치는 영향       |
|:-----------------|:----------------------------------------------------|:----------------------------|
| **해상도**       | 기본 모니터 해상도 또는 일반적인 1080p/720p 설정으로 낮추기 | 디스플레이 모드 충돌 해결     |
| **표시 모드**    | 전체 화면에서 창 모드 또는 테두리 없는 창 모드로 전환 | 전체 화면 문제 해결에 도움   |
| **그래픽 품질**  | "낮음" 또는 "보통" 사전 설정으로 설정                | GPU 부하 감소, 충돌 방지     |
| **수직 동기화**  | 처음에는 비활성화하고, 다른 문제가 발생하면 활성화     | 화면 찢어짐 및 입력 지연 방지 가능, 때로는 충돌 발생 |
| **DirectX 버전** | 가능하다면 DirectX 11과 DirectX 12 사이에서 전환    | API 호환성 문제 해결        |

<Callout type="info" title="설정을 보지 않고 액세스">
게임을 볼 수 없다면 `Alt + Enter`를 눌러 창 모드로 전환해 보십시오. 그러면 그래픽 설정으로 이동할 수 있을 것입니다. 그렇지 않다면, 게임 설치 디렉토리에 있는 게임 구성 파일을 확인하여 수동으로 조정하십시오.
</Callout>

검은 화면은 시스템 하드웨어 또는 소프트웨어 구성에 더 깊은 문제가 있음을 나타낼 수도 있습니다.

### 과열되는 부품

과열되는 CPU 또는 GPU는 시스템 불안정을 유발하여 충돌 또는 검은 화면으로 이어질 수 있습니다.
- **온도 모니터링**: HWMonitor 또는 MSI Afterburner와 같은 소프트웨어를 사용하여 게임 플레이 중 CPU 및 GPU 온도를 확인하십시오.
- **냉각 개선**: PC 팬이 깨끗하고 제대로 작동하는지 확인하십시오. 케이스 내부의 공기 흐름을 개선하는 것을 고려하십시오.

### 손상된 Windows 파일

시스템 파일 손상은 게임 성능 및 안정성에 영향을 미칠 수 있습니다.
- **SFC 스캔 실행**: 관리자 권한으로 명령 프롬프트를 열고 `sfc /scannow`를 입력하십시오. 이는 손상된 Windows 시스템 파일을 검색하고 복구합니다.
- **DISM 도구 실행**: SFC가 실패하면 명령 프롬프트(관리자)에서 `DISM /Online /Cleanup-Image /RestoreHealth`를 사용하여 Windows 이미지를 복구하십시오.

### 전원 공급 장치(PSU) 문제

불충분하거나 고장난 PSU는 특히 부하 시 전원 공급 문제를 일으켜 검은 화면을 유발할 수 있습니다.
- **PSU 와트 확인**: PSU가 모든 구성 요소, 특히 그래픽 카드에 충분한 와트를 제공하는지 확인하십시오.
- **연결 검사**: GPU 및 마더보드에 연결된 모든 전원 케이블이 단단히 연결되어 있는지 확인하십시오.

<CardGrid cols={2}>
  <Card title="그래픽 카드 확인">
    **증상**: 검은 화면, 시끄러운 팬 소음, 시스템 멈춤.
    **조치**: PCIe 슬롯에 제대로 장착되었는지, 전원 커넥터가 연결되었는지 확인하고, 가능하면 다른 GPU로 테스트하십시오.
  </Card>
  <Card title="메모리(RAM) 테스트">
    **증상**: 무작위 충돌, 검은 화면, 시스템 재부팅.
    **조치**: Windows 메모리 진단 도구 또는 MemTest86을 실행하여 RAM 스틱에 결함이 있는지 확인하십시오.
  </Card>
</CardGrid>
