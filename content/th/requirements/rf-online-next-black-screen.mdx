### การปรับการตั้งค่ากราฟิกในเกม

หากคุณสามารถเข้าเกมได้ครู่หนึ่งหรือได้ยินเสียงแม้ว่าหน้าจอจะดำ ให้ลองปรับการตั้งค่ากราฟิกของคุณ

| หมวดหมู่การตั้งค่า | การเปลี่ยนแปลงที่แนะนำ | ผลกระทบต่อจอดำ |
|:-----------------|:-------------------|:-----------------------|
| **ความละเอียด**   | ลดเป็นความละเอียดหน้าจอเนทีฟหรือการตั้งค่า 1080p/720p ทั่วไป | แก้ไขความขัดแย้งของโหมดการแสดงผล |
| **โหมดการแสดงผล** | สลับจาก Fullscreen เป็น Windowed หรือ Borderless Windowed | ช่วยแก้ปัญหา Fullscreen แบบพิเศษ |
| **คุณภาพกราฟิก** | ตั้งค่าเป็นค่าล่วงหน้า "ต่ำ" หรือ "ปานกลาง" | ลดภาระ GPU ป้องกันการขัดข้อง |
| **V-Sync**       | ปิดใช้งานในตอนแรก จากนั้นเปิดใช้งานหากเกิดปัญหาอื่น | สามารถป้องกันการฉีกขาดของหน้าจอและความล่าช้าในการป้อนข้อมูล แต่บางครั้งอาจทำให้เกิดความขัดแย้ง |
| **เวอร์ชัน DirectX** | หากมี ให้สลับระหว่าง DirectX 11 และ DirectX 12 | แก้ไขปัญหาความเข้ากันได้ของ API |

<Callout type="info" title="การเข้าถึงการตั้งค่าโดยไม่เห็น">

</Callout>

### ส่วนประกอบที่ร้อนจัด

CPU หรือ GPU ที่ร้อนจัดอาจทำให้ระบบไม่เสถียร นำไปสู่การขัดข้องหรือจอดำ
- **ตรวจสอบอุณหภูมิ**: ใช้ซอฟต์แวร์เช่น HWMonitor หรือ MSI Afterburner เพื่อตรวจสอบอุณหภูมิ CPU และ GPU ของคุณระหว่างการเล่นเกม
- **ปรับปรุงการระบายความร้อน**: ตรวจสอบให้แน่ใจว่าพัดลมของ PC ของคุณสะอาดและทำงานอย่างถูกต้อง พิจารณาปรับปรุงการไหลเวียนของอากาศภายในเคสของคุณ

### ไฟล์ Windows ที่เสียหาย

ความเสียหายของไฟล์ระบบอาจส่งผลกระทบต่อประสิทธิภาพและความเสถียรของเกม
- **เรียกใช้ SFC Scan**: เปิด Command Prompt ในฐานะผู้ดูแลระบบและพิมพ์ `sfc /scannow` ซึ่งจะสแกนหาและซ่อมแซมไฟล์ระบบ Windows ที่เสียหาย
- **เรียกใช้ DISM Tool**: หาก SFC ล้มเหลว ให้ใช้ `DISM /Online /Cleanup-Image /RestoreHealth` ใน Command Prompt (ผู้ดูแลระบบ) เพื่อซ่อมแซมภาพ Windows

### ปัญหาหน่วยจ่ายไฟ (PSU)

PSU ที่ไม่เพียงพอหรือทำงานผิดปกติอาจนำไปสู่ปัญหาการจ่ายไฟ โดยเฉพาะอย่างยิ่งภายใต้ภาระหนัก ส่งผลให้เกิดจอดำ
- **ตรวจสอบกำลังไฟ PSU**: ตรวจสอบให้แน่ใจว่า PSU ของคุณให้กำลังไฟเพียงพอสำหรับส่วนประกอบทั้งหมดของคุณ โดยเฉพาะการ์ดจอของคุณ
- **ตรวจสอบการเชื่อมต่อ**: ตรวจสอบว่าสายไฟทั้งหมดที่เชื่อมต่อกับการ์ดจอและเมนบอร์ดของคุณเชื่อมต่ออย่างแน่นหนา
