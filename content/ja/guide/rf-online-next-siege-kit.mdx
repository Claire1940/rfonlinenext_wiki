<Section icon={Shield} title="鉱山戦の戦術：RF Online Next シージキットの活用">

鉱山戦と勢力戦は、**rf online nextのシージキット**の究極の証明場所です。これらの大規模な戦闘は、中央の資源の支配と拠点インフラの防衛に焦点を当てています。これらの戦場独自のメカニズムにより、長距離かつ高火力の火力支援セットアップが非常に効果的になります。

勢力戦は拠点の採掘機を中心に展開します。これらの目標は純粋な攻撃力によってダメージを受けるのではなく、採掘機のHPは打撃回数（ヒット数）によってのみ減少します。配置されたランチャーは、安全な距離からこれらの構造物に連続して攻撃を降らせることができ、敵の防衛陣がカウンタープッシュを組織する前に素早く破壊することができます。

| 戦争の要素 | 出現時間 / トリガー | 採掘機への影響 | 戦略 |
| | | | |
| ホーリーストーンキーパー | 開始1分後 | 防御バリアを付与 | ラストヒットを最優先 |
| 飛行場 | 開始10分後 | 勢力間の側面奇襲を可能にする | シージラインでガードする |
| バリケード | 拠点入り口に有効化 | 直接的な地上ルートを遮断 | ランチャーの攻撃で破壊する |
| 採掘機 | マッチ中常に有効化 | ヒット数でHPが減少 | 高い攻撃速度で防衛/攻撃する |

<Callout type="success" title="採掘機のシールド">

</Callout>

</Section>

<Section icon={Sparkles} title="究極の遠距離ビルドの構築">

シージスタンス使用時のパフォーマンスを最大化するためには、適切な育成経路に投資する必要があります。ゲームの柔軟なバイオスーツ切り替えメカニズムにより、メインのダメージセットアップを運用しながら、サポート用ユーティリティへのアクセスも維持できます。物理出力を上昇させ、攻撃が標的に確実に命中するようなギアやステータスに焦点を当ててください。

| 育成レイヤー | システムの焦点 | 遠距離ビルドのメリット | 優先度 |
| | | | |
| 装備 | 強化＆プライムモッド | 基本物理ダメージの増加 | 高 |
| アドオン | アクティブスキルブースト | 遠距離スキルにクラウドコントロールを追加 | 高 |
| コレクション | 攻撃＆命中率 | 攻撃が安定してヒットするよう保証 | 高 |
| ローバー | チューニング＆ワークショップ | パッシブバフ、EXP獲得量、回復ユーティリティ | 中 |

まず、デモリッシャーやパニッシャーなどの物理ダメージバイオスーツの構築を優先してください。これらのスーツは、ランチャーの重い一撃を真に壊滅的なものにする純粋なダメージ補正を提供します。装備強化メニューからメイン武器を強化し、コレクションマイルストーンでは「攻撃力」と「命中率」のステータスに焦点を当ててください。命中率は極めて重要です。シージのボレー中にターゲットを外すと、貴重なクールダウン時間を無駄にし、カウンター攻撃にさらされる原因になります。

<Callout type="info" title="ローバーの同期">
物理攻撃速度と回復アイテムの所持容量にパッシブボーナスを提供するローバーを装備しましょう。このセットアップは、サポートクラスが他の場所で手一杯になっている際の長いシージ固定期間中にキャラクターを維持するのに役立ちます。
</Callout>
