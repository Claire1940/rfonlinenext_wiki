| **主要ボス** | ホーリーストーンキーパー（戦場の中心） |

<Section icon={Target} title="派閥の目標とエクスカベーターの防衛">

各派閥（ベラート、コラ、アクレシア）は、**RF Online Next チップウォー**開始時に、それぞれの基地内に設置された一連のエクスカベーターからスタートします。これらの機械は、派閥の経済と名誉の生命線です。それらを保護するには、固定防御と積極的な偵察の組み合わせが必要です。

戦場は守護者とバリケードによって強化されています。自派閥が中央を支配すると、基地のタレットが起動し、侵入者に対して大規模な防御上の優位性を提供します。

| 目標タイプ | 機能 | ヒットメカニズム |
| :--- | :--- | :--- |
| **エクスカベーター** | 主要な勝利目標 | ヒット数に基づく減少 |
| **バリケード** | 基地の入り口をブロック | 高い耐久性、タレットの射撃が必要 |
| **ガーディアンNPC** | 自動基地防御 | 標準HP、時間とともにリスポーン |
| **タレット** | 防御射撃支援 | ホーリーストーンキーパーのキルによって起動 |

<Callout type="warning" title="重要な警告">
基地のバリケードを無視しないでください。バリケードが崩れると、敵派閥が聖なる兵器でエクスカベーター室に殺到し、防御がほぼ不可能になります。
</Callout>

<Section icon={Zap} title="ホーリーストーンキーパーと戦場のバフ">

ホーリーストーンキーパーは、**RF Online Next チップウォー**の中心的なボスです。戦争開始から1分後にマップ中央にスポーンします。このエンティティは単なる戦利品のターゲットではなく、「最終ヒット」を与えた派閥に圧倒的な力をもたらす戦術的なツールです。

<Steps>
  <Step num="1" title="初期交戦">
    戦争開始後すぐにマップ中央へ移動します。最初のスポーンは1分マークで行われます。
  </Step>
  <Step num="2" title="最終ヒットプッシュ">
    派閥のバーストダメージを連携させます。最終ヒットを与えた派閥のみがホーリーストーンキーパーの祝福を受けます。
  </Step>
  <Step num="3" title="バフの蓄積">
    バフはキルが成功するごとにスケールアップします。移動速度、攻撃力、防御力、命中率、回避率が複数のティアで向上します。
  </Step>
  <Step num="4" title="タレットの起動">
    キルを確保すると、自派閥のタレットが起動し、敵のバリケードを自動的にターゲットして新しい攻撃経路を開きます。
  </Step>
</Steps>

| バフティア | ステータス増加 | 副次効果 |
| :--- | :--- | :--- |
| **ティア1** | 攻撃力/防御力 +5% | 小程度の速度ブースト |
| **ティア2** | 攻撃力/命中率 +10% | 中程度の回避ブースト |
| **ティア3** | 全戦闘ステータス +20% | 大規模な速度と回避 |

<Callout type="info" title="ボスのリスポーン時間">
ホーリーストーンキーパーは倒されてから3分後にリスポーンします。成功した派閥は、敵が次のバフサイクルを奪うのを防ぐために中央のコントロールを維持する必要があります。
</Callout>

</Section>

<Section icon={Shield} title="PvPにおける聖なる兵器戦略">

聖なる兵器は、**RF Online Next チップウォー**における究極の戦場ツールです。これらは標準装備ではなく、膠着状態を打破したり、防御を数秒で破壊したりできる限定使用の資産です。MAUを操縦するタイミングやランチャーを配備するタイミングを理解することが、失敗した襲撃と成功したチップ破壊の違いを分けることになります。

| 聖なる兵器 | 戦場での役割 | 最適な使用シナリオ |
| :--- | :--- | :--- |
| **MAU** | 操縦される最前線の戦車 | 敵のエクスカベーター室への突撃 |
| **ランチャー** | 重火力支援 | 敵の退却ルートを妨害 |
| **アニムス** | 召喚されたコントローラー | ボス戦中に敵の再配置を強制 |

**聖なる兵器展開チェックリスト：**
- **タイミング**: ホーリーストーンキーパーのHPが10%になったとき、または敵のバリケードを突破するときに展開を温存します。
- **位置取り**: ランチャーは高台または味方の最前線ユニットの後ろに配置する必要があります。
- **連携**: MAUを単独で展開しないでください。常に修理と支援のために**テクニシャン**バイオスーツが近くにいることを確認してください。

<Callout type="tip" title="プロのヒント">
「スティールウォッチャー」の実績には、聖なる兵器の展開が5回必要です。最終的なハイステークスプッシュの前に、戦争の初期段階で展開のタイミングを練習してください。
</Callout>

</Section>

傭兵システムでは、ギルドマスターがソロプレイヤーを募集して戦力を増強できます。このシステムには、参加と勝利に基づいて分配される100から100,000ダイヤのダイヤモンド賞金プールが含まれています。

| 傭兵ルール | 詳細 |
| :--- | :--- |
| **募集締め切り** | 21:10 (サーバー時間) |
| **賞金確定** | 21:25 (ダイヤモンドプールへの変更なし) |
| **報酬範囲** | 100～100,000ダイヤ |
| **資格** | 報酬処理まで派閥に留まる必要があります |

<Callout type="warning" title="飛行リスク">
長時間高すぎると燃料が急速に枯渇します。空中での燃料不足は、即座の墜落死につながり、最終的なエクスカベータープッシュの際に位置を失うことがよくあります。
</Callout>
